# ☁️ CloudMind AI

An offline AI-powered Cloud Architecture Assistant built using FastAPI, Ollama, FAISS, and React.

## 🚀 Features (Planned)

- Local SLM using Ollama
- RAG Pipeline
- PDF Upload
- Semantic Search
- FAISS Vector Database
- FastAPI Backend
- React Frontend
- Docker Support
- AWS Deployment

## 🏗 Tech Stack

- Python
- FastAPI
- Ollama
- Microsoft Phi-4 Mini
- FAISS
- React
- Docker
- AWS
